打开浏览器，内存中就会有这几个对象 BOM 浏览器对象模型 (Browser Object Model) Window 窗口对象 ------------------------------------------ Navigate 和浏览器版本相关的 * userAgent 浏览器的相关信息 Screen 和屏幕相关的对象 History 和浏览器历史相关 * back() 返回到上一个页面 * forward() 去下一个页面 * go() 例如: go(1)等于forward() go(-1)等于back() Location 和浏览器地址相关的对象 * href 获取和设置浏览器的路径 Document 文档对象
