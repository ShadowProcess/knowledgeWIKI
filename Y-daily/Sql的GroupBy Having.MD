#注意(请仔细理解这句话)：
#因为having的作用是对使用group by进行分组统计后[内存中的结果]进行进一步的筛选。
#而不是对group by进行分组统计后[展示到你屏幕上的结果]进行进一步筛选。 
```
___________
|id |name|
|__________
|1	| a  |
|__________
|2	| a  |
|__________
|3	| b  |
|__________
|4	| c  |
-----------
```

```sql
select * from `abc` group by `name`;  
```
```
___________
|id |name|
|__________
|1	| a  |
|__________
|3	| b  |
|__________
|4	| c  |
-----------
```




```sql
select * from `abc` group by `name` having count(name) > 1;
```
```
____________
| id | name|
------------
| 1  |  a  |
------------
```
